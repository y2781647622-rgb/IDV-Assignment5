SVG1 制作过程详解

几何图形动画的完整创建流程

1 创建基础画布

10x10 画布

步骤1效果：基础画布框架

首先创建一个10x10单位的SVG画布，设置合适的viewBox和尺寸。这一步为整个图形提供了绘制空间。

关键点：
  • 确定合适的坐标系和比例
  • 设置viewBox为0 0 10 10
  • 为后续图形定位打下基础
<svg xmlns="http://www.w3.org/2000/svg" width="100%" height="100%" viewBox="0 0 10 10"> <!-- 白色背景 --> <rect x="0" y="0" width="10" height="10" fill="white"/> </svg>

2 绘制平行四边形

步骤2效果：平行四边形轮廓

使用polyline元素绘制平行四边形，设置四个关键点的坐标，并添加标签标识各个顶点。

坐标说明：
  • A(1,1) - 红色点
  • B(1,4) - 绿色点
  • C(3,6) - 橙色点
  • D(3,3) - 紫色点
<!-- 主多边形 --> <polyline id="abcd" points="1,1 1,4 3,6 3,3 1,1" fill="none" stroke="#000" stroke-width="0.08" stroke-linecap="round" stroke-linejoin="round"/> <!-- 标签 --> <g font-size="0.35" fill="#000" text-anchor="start" dominant-baseline="middle"> <text x="0.5" y="1">A</text> <text x="0.5" y="4">B</text> <text x="3.2" y="3">D</text> <text x="3.2" y="6">C</text> </g>

3 连接对角线并添加箭头

步骤3效果：添加对角线

绘制两条对角线并添加箭头指示方向，使用marker元素定义箭头样式，并设置虚线效果。

对角线说明：
  • DA线：从D(3,3)到A(1,1)，蓝色虚线箭头
  • CB线：从C(3,6)到B(1,4)，黑色虚线箭头
  • 使用marker-end属性添加箭头
<!-- 定义箭头标记 --> <defs> <marker id="arrow-blue" viewBox="0 0 10 10" refX="10" refY="5" markerWidth="4" markerHeight="4" orient="auto"> <path d="M0 0 L10 5 L0 10 Z" fill="#1976d2"/> </marker> </defs> <!-- 连接线 --> <line id="DA" class="line-animation" x1="3" y1="6" x2="1" y2="1" stroke="#1976d2" stroke-width="0.08" stroke-dasharray="0.2 0.2" marker-end="url(#arrow-blue)"/>

4 添加动画和交互效果

步骤4效果：动画和交互

使用CSS和JavaScript添加动画效果和交互功能，包括虚线流动动画、标签闪烁效果和点击交互。

动画效果：
  • 主多边形虚线流动动画
  • 标签闪烁效果
  • 对角线虚线流动
  • 点击点高亮相关线条
<!-- CSS动画 --> <style> #svg1 .line-animation { animation: pulse 2s ease-in-out infinite; } @keyframes pulse { 0%, 100% { opacity: 1; } 50% { opacity: 0.7; } } </style> <!-- JavaScript交互 --> <script> document.addEventListener('DOMContentLoaded', function() { const points = document.querySelectorAll('.point'); points.forEach(point => { point.addEventListener('click', function() { const pointName = this.getAttribute('data-point'); highlightConnectedLines(pointName); }); }); }); </script>
