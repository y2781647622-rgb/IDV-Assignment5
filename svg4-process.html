SVG4 制作过程详解

探索概率密度函数图表的完整制作流程

1

创建画布并建立坐标系

首先创建一个900x700像素的SVG画布，并在其中建立直角坐标系。坐标系的范围是X轴从-2到5，Y轴从0到0.35。这个坐标系将用于绘制概率密度函数曲线。

<!-- 创建SVG画布和坐标系 -->
<svg xmlns="http://www.w3.org/2000/svg" width="450" height="600" viewBox="0 0 900 700">
  <g id="plot4" transform="translate(180,170)">
    <rect x="0" y="0" width="550" height="380" fill="#ffffff" stroke="#e5e7eb" stroke-width="2"/>
  </g>
</svg>
2

建立X轴和Y轴

在画布上添加完整的X轴和Y轴，包括刻度线和标签。X轴表示变量x的取值范围，Y轴表示概率密度值。添加网格线以便更准确地读取数值。

<!-- 添加坐标轴和刻度 -->
<g stroke="#111827" stroke-width="2">
  <line x1="0" y1="380" x2="540" y2="380"/>
  <polygon points="540,380 520,370 520,390" fill="#111827"/>
  <line x1="0" y1="380" x2="0" y2="0"/>
  <polygon points="0,0 -8,14 8,14" fill="#111827"/>
</g>
3

绘制黄色和青色概率密度曲线

基于概率密度函数绘制两条曲线：黄色曲线表示p(x, y=0)·ν(x)，青色曲线表示p(x, y=1)·ν(x)。这两条曲线通过数学函数计算得到，并添加了渐变效果和动画。

<!-- 绘制概率密度曲线 -->
<path id="curve0" fill="none" stroke="#eab54f" stroke-width="3.5" opacity="0.98"/>
<path id="curve1" fill="none" stroke="#1f8aa6" stroke-width="3.5" opacity="0.98"/>

<!-- JavaScript函数计算曲线点 -->
<script>
  // 计算黄色曲线
  function evalYellow(x) {
    const base = 0.22*Math.exp(-Math.pow((x-0.25)/1.2,2));
    const cross= 0.13*Math.exp(-Math.pow((x-1.1)/0.22,2));
    return Math.max(0, Math.min(0.35, Math.max(base,cross)));
  }
</script>
4

添加阈值虚线

在X轴的1位置左右两侧0.3处添加两条垂直虚线，表示决策边界。这些虚线有助于可视化分类阈值，并标记曲线相交的区域。

<!-- 添加阈值虚线 -->
<g stroke="#111827" stroke-width="2" stroke-dasharray="8 8" opacity="0.85">
  <line x1="189" y1="380" x2="189" y2="40"/>
  <line x1="231" y1="380" x2="231" y2="40"/>
</g>
5

计算并绘制重叠区域

计算两条曲线重叠的区域，并使用灰色半透明多边形填充这个区域。这个重叠区域表示两个概率密度函数同时具有显著值的区间，在分类问题中具有特殊意义。

<!-- 计算并绘制重叠区域 -->
<polygon id="overlap" fill="#6b7280" fill-opacity="0.45" stroke="none"/>

<!-- JavaScript计算重叠区域 -->
<script>
  // 阴影（真实重叠）区间
  const xL = -0.7, xR = 3, step = 0.02, top = [];
  for(let x = xL; x <= xR + 1e-12; x += step) {
    const y = Math.min(evalYellow(x), evalCyan(x));
    top.push([x2px(x), y2px(y)]);
  }
  top.push([x2px(xR), y0px]);
  top.push([x2px(xL), y0px]);

  document.getElementById('overlap').setAttribute('points',
    top.map(p => p[0].toFixed(1) + "," + p[1].toFixed(1)).join(" "));
</script>
6

添加标记和动画效果

最后，添加交点标记并使用CSS和JavaScript添加动画效果。包括曲线绘制动画、重叠区域呼吸效果、交点闪烁动画以及悬停交互效果，使图表更加生动和直观。

<!-- 添加交点和动画效果 -->
<circle id="xpointGlow" cx="213" cy="237" r="8"/>
<circle id="xpoint" cx="213" cy="237" r="3"/>

<!-- CSS动画 -->
<style>
  /* 曲线绘制动画 */
  #svg4 #curve0, #svg4 #curve1 {
    stroke-linecap: round;
    stroke-linejoin: round;
    animation: draw 30s ease-in-out infinite;
  }

  /* 重叠区域呼吸效果 */
  #svg4 #overlap {
    animation: breathe 3.2s ease-in-out infinite;
  }

  /* 交点闪烁动画 */
  #svg4 #xpointGlow {
    animation: blink 1.6s ease-in-out infinite;
  }
</style>
